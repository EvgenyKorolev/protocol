<- К оглавлению

Обозначение переменных для получения данных из конструктора протокола:

Переменные с данными из конструктора протоколоа:

  1. _cur_data - дата аттестата акркдитации (можно указать стиль отображения даты параметре style)
  2. _cur_data1 - дата аттестата акркдитации другой формат
  3. _cur_data2 - дата аттестата акркдитации другой формат
  4. _cur_data3 - дата аттестата акркдитации другой формат
  5. _cur_data4 - дата аттестата акркдитации другой формат
  6. _cur_engineer - инженер подписавший протокол (в формате И. И. Иванов)
  7. _cur_engineer1 - инженер подписавший протокол (в формате Иванов И. И. )
  8. _cur_engineer2 - инженер подписавший протокол (в формате Иван Иванов)
  9. _cur_engineer3 - инженер подписавший протокол (в формате Иванов Иван )
  10. _cur_engineer4 - инженер подписавший протокол (в формате Иванов Иван Иванович)
  11. _prt_number - номер протокола
  12. _prt_number_y - номер протокола с точкой и двумя цифрами года на конце
