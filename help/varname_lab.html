Обозначение переменных для получения данных о лаборатории:

Переменные с данными о клиенте:

  • Данные лоборатории:
    1. _lab_name - название лаборатории (ЮЛ)
    2. _lab_att - номер аттестата акредитации
    3. _lab_att_date - дата аттестата акркдитации
    4. _lab_att_date1 - дата аттестата акркдитации другой формат
    5. _lab_att_date2 - дата аттестата акркдитации другой формат
    6. _lab_att_date3 - дата аттестата акркдитации другой формат
    7. _lab_att_date4 - дата аттестата акркдитации другой формат
    8. _lab_inn - ИНН лаборатории
    9. _lab_email - email лаборатории
    10. _lab_full_name - полное наманование лаборатории
    11. _lab_dir_dol - название должности руководителя
    12. _lab_dir_name - ФИО руководителя организации
    13. _lab_labdir - ФИО руководителя лаборатории
  • Телефоны лоборатории:
    1. _lab_tel - телефон если есть
    2. _lab_tel1 - второй телефон если есть
    3. _lab_tel2 - третий телефон если есть
    4. _lab_tel3 - четвертый телефон если есть
    5. _lab_tel4 - пятый телефон если есть
  • Факсы лоборатории:
    1. _lab_fax - факс если есть
    2. _lab_fax1 - второй факс если есть
    3. _lab_fax2 - третий факс если есть
    4. _lab_fax3 - четвертый факс если есть
    5. _lab_fax4 - пятый факс если есть
  • Адрес почтовый:
    1. _lab_padr_f1 - адрес целиком
    2. _lab_padr_f2 - адрес целиком в другом формате
    3. _lab_padr_f3 - адрес целиком в другом формате
    4. _lab_padr_cont - страна нахождения лаборатории
    5. _lab_padr_statclass - вид региона лаборатории (республика, область)
    6. _lab_padr_state - название региона лаборатории
    7. _lab_padr_stdistr - название района региона лаборатории
    8. _lab_padr_citycl - вид населённого пункта лаборатории
    9. _lab_padr_city - назвние населённого пункта лаборатории
    10. _lab_padr_distr - район расположение (города) лаборатории
    11. _lab_padr_strcl - вид улицы лаборатории (проспект, тупик)
    12. _lab_padr_str - название улицы лаборатории
    13. _lab_padr_bnum - номер строения лаборатории
    14. _lab_padr_blit - литер строения лаборатории
    15. _lab_padr_ofnum - номер помещения лаборатории
    16. _lab_padr_oflit - литер помещения лаборатории
    17. _lab_padr_postind - почтовый индекс лаборатории
  • Адрес юридический:
    1. _lab_uadr_f1 - адрес целиком
    2. _lab_uadr_f2 - адрес целиком в другом формате
    3. _lab_uadr_f3 - адрес целиком в другом формате
    4. _lab_uadr_cont - страна нахождения лаборатории
    5. _lab_uadr_statclass - вид региона лаборатории (республика, область)
    6. _lab_uadr_state - название региона лаборатории
    7. _lab_uadr_stdistr - название района региона лаборатории
    8. _lab_uadr_citycl - вид населённого пункта лаборатории
    9. _lab_uadr_city - назвние населённого пункта лаборатории
    10. _lab_uadr_distr - район расположение (города) лаборатории
    11. _lab_uadr_strcl - вид улицы лаборатории (проспект, тупик)
    12. _lab_uadr_str - название улицы лаборатории
    13. _lab_uadr_bnum - номер строения лаборатории
    14. _lab_uadr_blit - литер строения лаборатории
    15. _lab_uadr_ofnum - номер помещения лаборатории
    16. _lab_uadr_oflit - литер помещения лаборатории
    17. _lab_uadr_postind - почтовый индекс лаборатории
  • Адрес фактический:
    1. _lab_fadr_f1 - адрес целиком
    2. _lab_fadr_f2 - адрес целиком в другом формате
    3. _lab_fadr_f3 - адрес целиком в другом формате
    4. _lab_fadr_cont - страна нахождения лаборатории
    5. _lab_fadr_statclass - вид региона лаборатории (республика, область)
    6. _lab_fadr_state - название региона лаборатории
    7. _lab_fadr_stdistr - название района региона лаборатории
    8. _lab_fadr_citycl - вид населённого пункта лаборатории
    9. _lab_fadr_city - назвние населённого пункта лаборатории
    10. _lab_fadr_distr - район расположение (города) лаборатории
    11. _lab_fadr_strcl - вид улицы лаборатории (проспект, тупик)
    12. _lab_fadr_str - название улицы лаборатории
    13. _lab_fadr_bnum - номер строения лаборатории
    14. _lab_fadr_blit - литер строения лаборатории
    15. _lab_fadr_ofnum - номер помещения лаборатории
    16. _lab_fadr_oflit - литер помещения лаборатории
    17. _lab_fadr_postind - почтовый индекс лаборатории
